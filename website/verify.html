Serverless Template

This is a template showcasing the serverless model using AWS S3, Lambda, Cognito, IAM, and API Gateway. The template also uses Bootstrap and FontAwesome. The code is based on two sample projects by AWS Labs: Serverless Web Application Workshop and Serverless SAM Farm.

The documentation (i.e. setup instructions, modification details) and full source code for this site as well as the pipeline that powers it is available within the GitHub repository.
