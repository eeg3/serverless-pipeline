Serverless Template

This is a template showcasing the serverless model using AWS S3, Lambda, Cognito, IAM, DynamoDB, and API Gateway. The template also uses Bootstrap and FontAwesome by default.

This template is based on the Amazon Web Services Labs' Serverless Workshop: Serverless Web Application, but is more stripped down to be less of a demonstration and more to provide an simpler foundation to build upon. Use the template as a starting point to create something more unique by building on or modifying it.
